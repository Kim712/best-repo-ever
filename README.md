# GitHub Workflow Practice Repository

## 📌 Overview
This repository is created as part of my learning journey on Git and GitHub, but no only for them.  
It demonstrates how to work with branches, commits, and pull requests using the GitHub workflow.

## 🚀 Purpose
The goal of this project is to:
- Practice creating and switching branches
- Make changes and commit them locally
- Push updates to GitHub
- Understand how collaboration works in GitHub

## 🛠️ Technologies Used
- Git
- GitHub
- Markdown (.md)

## ▶️ How to Use
1. Clone the repository
2. Create a new branch
3. Make changes to files
4. Commit your changes
5. Push the branch to GitHub

## 📚 What I Learned
- How to create and manage branches
- How to commit changes with meaningful messages
- Basics of GitHub workflow
